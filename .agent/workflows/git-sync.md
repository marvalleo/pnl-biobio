---
description: Sincroniza los cambios con el repositorio de Git en modo turbo (add, commit, push)
---

// turbo-all
1. Ejecuta `git add .` para preparar los cambios.
2. Ejecuta `git commit -m "{{message}}"` para guardar los cambios con el mensaje proporcionado.
3. Ejecuta `git push` para subir los cambios al servidor.
